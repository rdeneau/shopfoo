# Shopfoo

This application is a demo project showcasing the SAFE functional architecture, with domain workflows based on pseudo algebraic effects.

The solution is based on the [SAFEr.Template](https://github.com/Dzoukr/SAFEr.Template).

## Run the application

Run the relevant command(s) in a console at the root of the solution:

```powershell
# Both Client and Server
dotnet run

# Server only
dotnet run Server

# Client only
yarn start
`````
